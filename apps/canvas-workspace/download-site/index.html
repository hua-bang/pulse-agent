安装

macOS 首次打开

  1. 下载 DMG，并把 Pulse Canvas 拖进 Applications。
  2. 从 Applications 打开一次；如果看到“Pulse Canvas Not Opened”，点击 Done。
  3. 打开“系统设置 > 隐私与安全性”，在 Security 区域点击“仍要打开”。
  4. 再次确认“打开”。后续通常可直接启动。

这是早期版本暂未使用 Apple Developer ID 签名导致的安全提示，不影响本地使用。
